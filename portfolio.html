IF-Charts

IF-Charts is a tool for flight simulator enthusiasts to get charts for their flight simulator, anytime, and anywhere with an internet connection.

It's built with javascript, css and html, and it's also opensource! You can find the Github repository here.


Simple keylogger for Windows, Linux and Mac

Simple keylogger is a program that logs your keystrokes. It's available for Windows, Linux and Mac, written in C++, Python and C respectively.

You can find it on Github here.


Simple HTML and CSS website template

The simple websote template is a html and css template for anyone that wants to get into making websites. It's got comments on both the html page and css page, and can be used as a boilerplate by and programmer, whether experienced or new.

Be sure to check out the repository here!
